## [x] 【UI渲染】應正確渲染管理後台的所有初始元素
**範例輸入**：設定 AuthContext 回傳預設 user (`role: 'admin'`)
**期待輸出**：頁面上應包含標題「🛠️ 管理後台」、返回 Dashboard 連結、用戶角色標籤 (管理員)、登出按鈕及頁面說明內容。

---

## [x] 【UI渲染】當使用者的角色不是 admin 時顯示一般用戶標籤
**範例輸入**：設定 AuthContext 回傳 user (`role: 'user'`)
**期待輸出**：角色標籤應顯示「一般用戶」而非「管理員」。

---

## [x] 【路由與導覽】點擊返回連結應導向至 Dashboard
**範例輸入**：點擊「← 返回」連結
**期待輸出**：路由導向至 `/dashboard`。

---

## [x] 【API與狀態互動】點擊登出按鈕應呼叫 logout 並導向至 Login
**範例輸入**：點擊「登出」按鈕
**期待輸出**：應呼叫 `logout` 函式，且路由導向至 `/login` (`replace: true`)。
